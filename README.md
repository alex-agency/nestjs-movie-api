### Running the app

1. Get repository:

   ```
   git clone --depth 1 https://github.com/alex-agency/nestjs-movie-api.git nest-cinema
   ```

1. Run docker:

   ```
   docker compose up -d
   ```

1. Review logs:

   ```
   docker compose logs
   ```

1. Browse app: <http://localhost:3000>
